Tentative Syllabus

Each Module should take approximately two weeks. You will complete a quiz and an assignment for each module. Expect an assessment each Friday, unless specified otherwise. You must be present for all assessments. If an asessment is missed due to a planned or foreseeable absence, it is your responsibility to arrange to complete the assessment ahead of time. In case of illness, you are expected to complete the assessment immediately upon return.

Course Outlines: ICS3C or ICS3U

0. File Maintenance and Version Control

We'll be using GitHub (link at top left) to organise our files. It lets you save different versions of a project as you go along, and keeps shared work organised when working on group projects. It'as also where you'll be accessing and submitting coding assignments, but we'll get to coding later.

  1. Say Create a GitHub page (like this one) by creating a gh-pages branch from your own Repository. We will use your github page to organise your JavaScript assignments later in the course. GitHub hosts your pages for free!
  2. If you want to get fancy with your READMEs, and generally be more stylish, look at Markdown Syntax, used to style .md files

1. Intro to HTML and CSS

We'll be using GitHub (link at top left) to organise our files. It lets you save different versions of a project as you go along, and keeps shared work organised when working on group projects. It'as also where you'll be accessing and submitting coding assignments, but we'll get to coding later.

  1. Get your site online by uploading files to gh-pages branch of your repository. See instructions in Module 0 if you're late joining.
  2. REVIEW HTML. Under "Responsive Web Design > Basic HTML": You should be comfortable with all topics up to and including "Create an ordered List".
  3. REVIEW CSS. Under "Responsive Web Design > Basic CSS": You should be comfortable with all topics up to and including "Size Your Images".
  4. If you're already familiar with these topics, feel free to explore further to make your website look even more professional.
