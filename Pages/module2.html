Module 2 (Oct 1-12)

This module will provide a cursory introduction to computer programming topics with an opportunity to become familiar with basic programming using scracth. Any material presented in this module is fair game for the quiz.

Counting, ASCII, Unicode

Based on the brief lecture in class, you should be familiar with various counting systems (unary, binary, decimal, hexadecimal) and their unique advantages and applications

Things to Know for Quiz

  • Know basic html tags and CSS
  • Be able to explain the differences between counting systems, including binary, decimal, and hexadecimal. Describe uses and advantages/disadvantages of each
  • Be able to convert binary numbers to decimal and decimal to binary
  • Be able to create a css rule that uses hexadecimal colour notation: E.g. #0000FF is Green
  • Be able to describe what ASCII is and why the need for unicode arose
  • You may also be asked to comment, in general terms, on your scratch game
